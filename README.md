# Arduino Obstacle Avoiding Robot

## Project Overview
Autonomous obstacle-avoiding robot car built on Arduino Uno platform.
- **Microcontroller:** Arduino Uno (ELEGOO UNO R3)
- **Sensor:** HC-SR04 Ultrasonic
- **Motor Driver:** L298N
- **Chassis:** Durashmall 2WD with dual DC gear motors
